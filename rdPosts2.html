以RF，MG，AR为输出核心的常见模板队伍与改装简述

2019-03-17 21:39

极主观内容警告！
首先必须要指明几点

1.我配队的时候不会每次都想那么多东西↑。

2.实际过图的时候，我会拿个我觉得合适的过去队伍当模板改来改去。 可能改到爹妈不认

3.极限合理的配队没有实际意义。能达成配队目的我就不会再改了，哪怕这个队伍在输出/防御方面可能还有改进余地，哪怕属性溢出让别人看着很难受。

RF相关：
RF队最近定位正在变得单调起来……
常见的有3套模板。

模板1：1火力HG + 1射速HG + 1防御性HG + 1火力RF + 1射速RF
比较传统的一种RF模板。
典型代表是大家常说的李白维绿灰。




稍微说一下李白维绿灰的队伍细节

  • WA2000的光环提供给维尔德，使其早点开技能。


  • WA2000和李恩菲尔德在基础射速差距和射速光环差距配置下，初始射击间隔达到6帧，以前还有压李恩菲尔德射速强化的做法，可以极大减少鞭尸现象。


  • WA2000技能期射速为124




改装思路：

  • 目前环境下，军方和白色势力的主要伤害来源，都不太适合HG硬抗，所以思路一般为以攻为守或者借助妖精保护，所以防御性HG经常会被换下去——或者换成攻守兼备P22/海盗改


  • 1火力，1射速的HG/RF配置是基于旧铁血的血量水平做的搭配，新环境下敌人血量变化很大，注意火力和射速的调整来提升击杀效率。


  • 敌人血量和装甲越来越厚，鞭尸和伤害溢出的影响在淡化，虽然多多少少还是会带来影响


  • 暴击率不满终究还是个不稳定因素，对击杀效率比较执着的玩家建议满暴。


  • 以后可能RF夜战打闪情况变多，命中buff需要一定程度的重视。


模板2：暴击率提升手段+风暴
最常见的暴击率提升手段是搭配57，俗称貂暴队。


20%暴击光环+暴怒技之类的也可以。
其实就是模板1发展过来的。
剩下一个位置参考模板1的改装思路，看着放吧。




模板特点：
打靶输出比较高。
风暴开技能的时候要回到2号位。




模板细节问题：
57和风暴吃的CDbuff不同的话，长时间战斗中，两者技能触发时间点会错位。
如果吃全光环，那么两个RF很难错开射速，鞭尸情况会比模板1严重一些。
因为暴击单发非常高，所以道中打杂兵伤害溢出也相对严重些




改装思路：
单纯的打靶输出比较高的队伍，单纯得以至于我没什么好说的。

模板3：卡姐+蟒蛇+风暴
俗称卡蟒队(初始名为卡蟒暴)




队伍构成和运行机制：
卡姐触发被动，卡姐被动触发蟒蛇被动。
在卡姐被动覆盖率良好的情况下，可以给队伍大量射速buff和暴击buff，风暴将溢出的暴击buff转化为爆伤。
为了卡姐的被动覆盖率，5号位通常会选择光环技能双加速的M950A，当然也可以选别的。
7号位放M14改，FN49改，李恩菲尔德什么的都行，因为阵容射速加成较多，所以不太适合WA2000和K31。
因为射速和暴击都是平A特化buff，所以这个阵容和卡妹没有必然的联系。




队伍特点：
3种模板中，站桩打靶输出最高的一种。
卡姐再就业。
本质上接近4保1。
卡姐不能动，7号位不能动，蟒蛇一般不能动，风暴开技能的时候要回到2号位，总体来讲是个走位灵活性比较差的阵容，在卡抬手和轮流承伤等操作方面比较僵硬。
仍然是鞭尸略严重的队伍。
蟒蛇可以帮忙补一些输出，在夜战清杂兵相对有优势。




改装思路：
没什么好改的，基本定死的。
PS：阵型模拟器卡姐技能有bug，所以实际上卡姐没有被动，已于3月15日修复。

MG相关：
有两个模板，两个模板间没有明显的界限区分。

1.一波流模板：
3首轮MG+1HG+SG
或
2首轮MG+2HG+SG




队伍特点：
追求一波杀干净对方所有敌兵，以为攻守的队伍

MG方面，需要首轮能打出技能的MG，来保证首轮爆发力度，基本锁定暴动宣告，连珠终结，猎杀冲动，战地魔术这四种。

HG主要提供光环，技能可以不做考虑，主要考虑伤害buff，其次暴击buff和命中buff，射速buff几乎没用，

SG，属于万一没清干净的保险单位，大概。




队伍细节：

  • HG虽然说主要提供光环，不过如果开局走动时间略长以及MG弹链较长，还是可以吃到HG技能的。


  • 猎杀冲动效果受开局走动时间影响非常大(你也可以选择开局拖动等前置CD)


  • FG42面板太差，所以不怎么样。


  • 连珠终结那发高倍率子弹不可暴击。


  • 暴动宣告那发技能子弹可以二次暴击。


  • 17位的SG光环更适合2HG+2MG。


  • 4位最佳的HG为纳甘改，可以使2MG队伍理论输出值达到3MG水平。


  • 2MG鞭尸和伤害溢出会比3MG更为严重，不过2HG可以代替SG抗下一些SG无法抗住的高伤害攻击，或者低命中破甲技能，同时因为是3T组合，卡抬手的操作空间也相比较大。





改装思路：
1.强化输出:把SG下了换成MG/HG，打极限一波流，也就是所谓的打靶队。
2.强化防御:一般就是适量叠护甲，向铁甲队模板改装。
如果发现一个SG抗住不/无法吸引全部火力，可以上两个SG。

2.铁甲队模板：
3叠甲MG+1HG+SG




队伍特点：

  • 护甲达标之后非常能抗，输出环境相对安逸。


  • 不过盾甲妖精技能持续20s，20s之后可能护甲无法达标，输出压力还是有的。


  • 火压技能和护甲提升技能可以延长护甲达标时间。


  • 部分叠甲机枪，2轮开始技能发力，多轮输出要超过首轮MG





改装思路：
进一步提升防御：
护甲达标后防御手段就是贴膜了，SG提升闪避不靠谱。
护甲仍未达标的话，参考3.15护甲属性中的“如何使护甲达标”
如果发现一个SG抗住不/无法吸引全部火力，可以上两个SG

进一步提升伤害：
远高于敌方伤害的护甲是没有用的，如果护甲过多，可以换掉部分输出低的叠甲MG，比如将MG4换为MG5。

AR相关：
AR因为技能繁多，泛用性较强，阵容变化灵活，
所以不做常见模板的陈述，直接进行选择性陈述。
相关传送门：[[未完工][鸽了7个月]AR搭配指南]——作者[咖啡幻想]

AR队一般有两种搭配选择方式。

一.确定4号位人选，再围绕4号位进行搭配后排人选。
常见的4号位选择有两类：
1.利贝罗勒/HG
2.M4A1。




1.利贝罗勒和HG为4号位
两者在4号位效果相似。
利贝罗勒对输出BUFF能力超过HG，但是受走位限制，需要走位的话一般不会选择利贝罗勒。
利贝罗勒与HG主要搭配一些高爆发的AR，比如G11、AN94、闪电(夜)等，当然G41之类的其他AR也可以使用，不过97这种一般不会考虑。
此类阵容，一般6s爆发力度非常强，不过6s前输出略微缺乏，爆发期过后真空期也较长，技能覆盖率一般也偏低，而且最强力的爆发打手G11最近水土不服。




2.M4A1为4号位
这个是个人最近活动用的最多的AR阵容。
一个原因是常见的强力地图优势人形AR15、AN94、AK12在此阵容内兼容性都很良好。
另一个原因是随着敌方血量增厚与护甲提升，M4炮击效果逐渐变得突出起来。
而且相比血包/HG爆发阵容，前期输出更高，持续力也更加强劲。




前排选择：
buff搭配就不说了，看前面基础部分。
这里主要说一些枪种选取思路。

  • 首先，上手枪前排配合卡抬手等操作，让输出直接秒掉对面是最理想的，11章普通+紧急的敌人除了坦克和红雷都可以这样处理。


  • 如果自己输出力度没法满足条件，手枪会受损，先考虑战损能不能接受，不能接受就考虑换个SMG上去，可以考虑UMP45，CMS等这种对于队伍来讲攻守兼备的SMG，考虑hg许多战损来自于杂兵擦伤，也可以考虑投掷类副t强化对兵堆输出。


  • 如果对前排压力略大，比如BOSS战，强无敌战，硬接狙娘等，就考虑上防御性T，比如RO635，PM06，MP7，以及闪光弹力场盾等等，2SMG还是SMG+HG看压力情况，也可能使用嘲讽/克洛伊来代替人形保护队伍，具体情况具体决定。




一些针对性较强的情况，比如

  • 面对如白色光头等大量低攻无/低破甲的敌人，可能会考虑16哥或者sg，也可能是投掷类副t。


  • 面对红雷，可能会上烟雾弹、闪光弹、力场盾。


  • 面对木星炮，可能会上力场盾。


  • 面对编制攻击，可能会增加队伍贴膜手段。


  • 面对极高单发低频/接敌即重创的敌人，可能会把smg换回hg。



注意都是可能，以上“可能”都来自经验，经验导向的配队方式有何种问题我已经在拓展内容中谈过了。


具体到实际情况中的话，军方和白色势力敌人的主要战损来源都不适合人形硬抗，并且单编制血量和护甲偏高，所以都会倾向于HG前排，尤其是伤害buff类的HG。

二.确定前排人选，再围绕前排进行搭配。
此类情况下的前排人选——旋风
老实讲，不怎么想说这个……

旋风相关配队旋风自然是重点，因为他的输出能力发挥条件略苛刻，需要特意创造环境。
但不要忘记配队最终要以整个队伍的输出来做评估，牺牲队伍总输出来抬高旋风输出或者纠结旋风是否为MVP属于本末倒置。




旋风输出方面的特点：

  • 通过合适的搭配，有旋风的AR队伍，比同水平非M4A1 or血包4号位的输出能力要强些。


  • 掉小人会影响输出，旋风掉小人的风险较高。


  • 技能前置短，覆盖率高


  • 带光瞄打无闪单位输出大概在G11~AK12之间。


  • 因为SMG模板的问题，带光瞄打有闪单位输出会跳水，需要更换瞄具，且对命中buff需求较大。


  • 对爆伤类buff利用率较低。


  • 开启技能后单发伤害很高。


  • 能良好利用后排提供的高质量输出光环。




旋风队伍相关定位与队伍组成：
从上述特点，旋风队定位大致有两种

1.快速爆发清场
常用的单位有G36改，AUG，AK12，K2，MDR等前期输出较高且带有输出光环AR。
AK12作为独一无二的7号位能为2号位提供buff的人形，往往是7号位的优选人形。
从快速爆发清场这个角度来讲，你塞pkp其实也可以…… 最后改装得到一个一波流MG队

剩余一个前排单位一般为能buff到2147位的HG，如果是小额擦伤连斩的话，也可以考虑换SMG。
要注意保证旋风的命中。
因为夜战旋风不得不携带夜视仪，命中很低，所以效果一般般。

PS：因为现在人形光环和技能越来越花里胡哨，所以旋风也不见得一定在2号位，可能在5号位，然后拉到别的地方，或者站撸直接打死。
至于非技能期走位，旋风非技能期输出偏低，个人感觉无伤大雅。



2.打击装甲单位
常用的单位有闪电(夜)，G36改，AK12，MDR等能对装甲打出输出且带有输出光环的AR。
队伍里一般都要配备火号HG。
旋风的位置可以由CMS替换,CMS与旋风打甲输出谁高取决于buff情况与敌方装甲值。




不利于队伍发挥的场合有以下两种情况：
1.夜战打高强度闪避敌人

2.敌人无法在接敌前就消灭，且敌人无法被主T全部吸引/主T压力过大，需要第二个抗压T

混编队相关
混编队就没什么模板了，拿出来的混编队都是一种针对性很强的队伍。

混编队的本质就是由某些人形的特质来满足队伍特定的需求，这是一个质变。
混编队往往会放弃SMG与AR之间，MG与SG之间等光环，导致队伍总体输出有所下降，这是一个量变。
量变会导致质变，某些特质是由数量基础为支撑的。

对人形、各类技能、各类buff、敌方情况的细节和特性的了解程度，以及配队思维的灵活性决定了能否满足质变需求。
人形、装备、妖精、重装等基础练度，决定能否满足量变需求
